## 可以使用简单的表达式设置元素的样式, 使用calc关键字, 如下:
```javascript
// html
/*
<style>
	.page {
		width: 600px;
		height: 600px;
		background: red;
	}

	.box {
		width: calc(100% - 100px);
		height: 100%;
		background: blue;
	}
</style>

<div class="page">
	<div class="box">
	</div>
</div>
*/
// cale关键字可以进行简单的运算,  注意的是不兼容ie
