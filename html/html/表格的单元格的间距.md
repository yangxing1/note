## 表格的单元格的间距设置是在标签的属性上设置的， 如下：
```html
<table cellspacing="15">
<tbody>
<tr><td>hello</td><td>world</td></tr>
</tbody>
</table>
```

## 也可以在css 设置，如下：
```html
<style>
	/* 设置表格的边框 */
	table {
		border: solid #000;
		border-width: 1px 0 0 1px;
	}
	table td {
		border: solid #000;
		border-width: 0 1px 1px 0;
	}
	/* 消除表格单元格之间的距离 */
	table {
		border-collapse: collapse;
	}
</style>
<table>
	<tbody>
		<tr><td></td></tr>
	</tbody>
</table>
```
