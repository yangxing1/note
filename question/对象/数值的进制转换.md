## 一般的数值都是10进制, 数值变量的toString()方法可以将数值从10转换到其他进制, 以字符串的方式返回, 如下:
```javascript
var number = 3;
var number_2 = number.toString(2);	// '11'
```

## 可以使用parseInt()方法给数字或字符串转换进制, 如下:
```javascript
var number = 11;
var str = '11';
console.log(parseInt(number, 2));	// 11
console.log(parseInt(str, 2));		// 11
```
