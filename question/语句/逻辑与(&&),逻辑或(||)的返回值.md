## 需要注意的是&& 和 || 表达式并不一定会返回布尔值,而是根据操作数返回值

```javascript
// 逻辑与
console.log('a' && '');	// ''
console.log('' && 'a'); // ''
console.log('a' && 'b');	// 'b'
// 逻辑与表达式返回值的规则是先判断左操作数,如果为真值判断右操作数,不管右操作数为真或为假,都返回右操作数,如果左操作数为假就返回做操作数.

// 逻辑或
console.log('a' || ''); // 'a'
console.log('' || 'a'); // 'a'
//逻辑或表达式返回规则是先判断左操作数,如果为真直接返回左操作数,不管右操作数,如果左操作数为假,则判断右操作数,不管为真或为假都返回右操作数.
```
